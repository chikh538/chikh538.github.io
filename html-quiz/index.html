INLINE ELEMENT & A BLOCK-LEVEL ELEMENT

An inline element does not start on a new line and does not take up the full width of a page, only the space bounded by its opening and closing tag. Some examples are the anchor tag and image tag. A block-level element always starts on a new line and takes up the full width of a page, from left to right. Examples include heading tags, blockquote tags, and list tags.
